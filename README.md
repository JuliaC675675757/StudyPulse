# StudyPulse

## Project Description
StudyPulse is an Android application that helps students organize their study tasks and manage study sessions using a Pomodoro timer.

## Features
- User Login
- Dashboard
- Task Management
- Study Session Timer
- User Profile
- Logout

## Requirements
- Android Studio
- Android SDK
- Java 8 or higher
- Android Emulator or Android Device

## How to Set Up

1. Clone the repository.
2. Open the project in Android Studio.
3. Wait for Gradle Sync to finish.
4. Connect an Android device or start an emulator.
5. Click the Run button.

## How to Run

1. Launch the application.
2. Log in.
3. Navigate through Dashboard.
4. Add tasks.
5. Start a study session.
6. View the Profile page.
